Запусти полный набор тестов проекта и проанализируй результаты.

## Шаги

1. **Backend тесты:** выполни `cd backend && npx jest --verbose --no-cache`
2. **Frontend тесты:** выполни `cd frontend && npx vitest run --reporter=verbose`
3. **Анализ результатов:**
   - Выведи сводку: количество пройденных/упавших тестов для каждой части
   - Если есть упавшие тесты — проанализируй вывод ошибок и предложи исправления
   - Если все тесты прошли — подтверди что всё работает корректно

## Формат вывода

Выведи краткую таблицу результатов:
- Backend: X passed / Y failed (Z test suites)
- Frontend: X passed / Y failed (Z test suites)
